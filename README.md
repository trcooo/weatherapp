# WeatherApp (Python + HTML/CSS/JS)

Современное погодное веб‑приложение (white/blue).

## Локальный запуск

```bash
python -m venv .venv
# Windows: .venv\\Scripts\\activate
source .venv/bin/activate
pip install -r requirements.txt

# переменные окружения
cp .env.example .env
# заполните OPENWEATHER_API_KEY

python app.py
```

Откройте: http://localhost:8000
